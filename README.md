# Blacksmith

## Installation

Install dependencies

```
npm install
```

### Development dependencies

- browser-sync
- del
- gulp
- gulp-cli
- gulp-minify
- gulp-pug
- gulp-sass
- gulp.spritesmith
- merge-stream

## Execution

Compile views and assets into development folder and run local server

```
gulp dev
```

Compile views and assets into distribution folder

```
gulp dist
```
